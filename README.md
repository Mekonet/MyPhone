## Overview
MyPhone is a web application built with Angular.

## Features
- Create new records
- Update existing records

- Structured project architecture

The application follows Angular best practices:
- Components for UI
- Modular file structure
- TypeScript logic separation

## Installation & Run

1. Clone the repository
2. Run: npm install
3. Run: ng serve
4. Open: http://localhost:4200

This project was created to practice Angular fundamentals and front-end development principles.
